Pasos para iniciar el proyecto de prueba

Antes de iniciar el proyecto, descargar por git el repositorio https://github.com/izofer/pruebapi.git
despues de descargar, correr el comando por consola: composer update.

  1. 1 - Inicar el proyecto Laravel con el servidor que se desea seleccionar (Wamp, Laragon, o con el comando: php artisan serve),
  2. 2 - Crear una base de datos llamada: pruebapi
  3. 3 - Correr las migraciones de laravel con el comando: php artisan migrate --seed
  4. 4 - En Login, en la caja de texto api rest, escribir la URL generada por el servidor del proyecto, tener en cuenta el puerto.
  5. 5 - Iniciar sesión con los datos escritos por defecto del usuario administrador, estos datos se encuentran ubicados en Login.
  6. 5 - Solo el administrador puede editar, eliminar y ver la lista de usuarios.
  7. 6 - Los usuarios operadores solamente podrán editar su información.
  8. 7 - Cualquier duda sobre el proyecto comunicarse con Yefferson Guarnizo al célular 305 815 9488
